Ovládání světel pro CaspiaTech
===

Aplikace je určena pro prezentační účel. Demonstruje použití React framework a napojení na API (REST a Websockets).
Na pozadí jsou použity technologie Redux, Redux-Saga s Immutable.js a reselect. Pro překlady používám react-intl.

Instalace aplikace
---
```
$ mkdir lights-controler && cd lights-controler
$ git clone https://github.com/keyBeatz/lights-controler.git .
$ yarn
```

Start aplikace
---

```
$ yarn dev
```

Poté otevřete ve svém prohlížeči adresu `http://localhost:3000` a aplikace se načte.
